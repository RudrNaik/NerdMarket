# NerdMarket
Developed for Iowa State's COMS 3090 Software Development Practices- NerdMarket provides users with an easy to use portfolio for their Pokemon, Yu-Ghi-Oh, and Magic The Gathering trading cards. 

# Features
- Large library of cards direct from Pokemon, Yu-Ghi-Oh, and MTG APIs to track prices that updates every day at 1AM CST. 
- Portfolio view that displays all of the user's cards and track their price fluctuations over time. 
- Chats for each category of card to help users find others to trade cards with, or discuss market trends.
- Card Scanning that allows you to scan a card directly from your phone's camera and then see that card's price seamlessly.
- Card Search that allows you to look up a card and see its current price for the day. 
- Admin features such as unlocking accounts, banning them, or helping moderate chats via promoting users to being moderators.
